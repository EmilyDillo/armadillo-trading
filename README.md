# Armadillo Trading Platform
Automated paper-trading cycle + always-on dashboard. See RUNBOOK.md (operations) and SUCCESSOR.md (AI operator briefing).
The dashboard publishes to GitHub Pages on every weekday run: `https://<your-username>.github.io/<repo-name>/`
Setup: Settings -> Pages -> Source: GitHub Actions. Settings -> Secrets and variables -> Actions -> add ALPACA_API_KEY and ALPACA_SECRET_KEY.
